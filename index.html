Kebijakan Privasi WADS Digital

Terakhir diperbarui: 16 Januari 2026

Selamat datang di WADS Digital. Kami sangat menghargai privasi Anda dan berkomitmen untuk melindungi data pribadi Anda. Kebijakan Privasi ini menjelaskan bagaimana kami mengumpulkan, menggunakan, dan melindungi informasi saat Anda menggunakan aplikasi kami.

1. Informasi yang Kami Kumpulkan

Kami mengumpulkan informasi yang diperlukan untuk menjalankan fitur utama aplikasi, antara lain:

  • Gambar dan Kamera: Aplikasi meminta akses ke kamera dan galeri untuk mengambil gambar resi paket guna mendeteksi nomor telepon dan nominal COD menggunakan teknologi OCR.
  • Nomor Telepon: Nomor telepon yang terdeteksi atau diinput secara manual hanya digunakan untuk memicu aplikasi WhatsApp di perangkat Anda.

2. Bagaimana Kami Menggunakan Informasi

Informasi yang dikumpulkan digunakan untuk:

  • Memudahkan proses input nomor telepon tujuan WhatsApp tanpa perlu menyimpannya di kontak.
  • Mendeteksi tagihan COD secara otomatis untuk mempermudah komunikasi kurir.
  • Personalisasi template pesan sesuai dengan pengaturan yang Anda buat.

3. Layanan Pihak Ketiga (Google Cloud Vision)

Untuk fitur deteksi teks via kamera, aplikasi ini menggunakan Google Cloud Vision API. Saat Anda mengambil foto menggunakan kamera dalam aplikasi, gambar tersebut dikirim ke server Google untuk diproses secara sementara guna mendeteksi teks. Kami tidak menyimpan gambar tersebut di server kami, dan penggunaan data oleh Google tunduk pada Kebijakan Privasi Google.

4. Keamanan Data

Aplikasi ini tidak menyimpan nomor telepon atau gambar resi di database online kami. Semua data yang diproses (seperti template pesan dan preferensi) disimpan secara lokal di perangkat Anda menggunakan Shared Preferences.

5. Izin Perangkat

Aplikasi ini memerlukan izin berikut:

  • Camera: Untuk mengambil foto resi secara langsung.
  • Storage/Gallery: Untuk memproses gambar yang sudah ada atau menerima kiriman (share attachment) dari aplikasi lain.
  • Internet: Digunakan khusus untuk memproses OCR melalui Google Cloud Vision.

6. Perubahan Kebijakan

Kami dapat memperbarui Kebijakan Privasi ini dari waktu ke waktu. Penggunaan berkelanjutan atas aplikasi ini setelah perubahan dianggap sebagai persetujuan terhadap kebijakan yang baru.
